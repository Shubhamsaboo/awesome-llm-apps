# Style Guide Reference

## Banned Words
revolutionary, synergy, cutting-edge, next-level, game-changing, leverage, unlock, transform, disrupt, innovative

## CTA Templates
- "Start [action] today"
- "Get [specific result] in [timeframe]"
- "Join [number] [people] who [achieved result]"

## Headline Formulas
- [Number] ways to [achieve desired outcome]
- How [company/person] [achieved result] in [timeframe]
- The [adjective] guide to [topic]
