# Context Kit — Personal AI Operating System

A skill system for giving AI agents persistent personal context. Eliminates context amnesia.

## What it does

4 Markdown templates (Personal Context Artifacts) encode who you are, how you decide, how you write, and your hard rules. Load them at session start → AI operates with full personal context from turn one.

## Quick start

```bash
# Install
curl -fsSL https://raw.githubusercontent.com/JDDavenport/context-kit/main/install.sh | bash

# Edit your templates
code ~/.context-kit/wiki.md
code ~/.context-kit/mental-models.md
code ~/.context-kit/voice.md
code ~/.context-kit/protocols.md
```
